===============================================================================

　UnrealRemoco / クライアント

===============================================================================


■起動方法

　UnrealClient.exe を実行して下さい。


■メニューバー

　アプリ / 終了

　　このアプリケーションを終了します。

　接続 / 接続

　　接続ダイアログを開きます。

　接続 / 再接続

　　接続中の場合、再接続します。

　接続 / モニタ選択

　　接続中、表示及び操作するサーバー側のモニタを変更します。

　接続 / 切断

　　接続中の場合、切断します。

　設定 / 画質

　　表示するサーバー画面の画質を設定します。
　　画質は以下のとおりです。

　　　0 ～ 100 = JPEG, 値が高いほど画質が良い。
　　　101      = PNG

　設定 / その他の設定

　　設定ダイアログを開きます。


■接続ダイアログ

　設定リスト（上部のリスト）

　　行をダブルクリック ... 選択された行の設定で接続を試みます。

　　右クリック
　　　接続           ... 選択された行の設定で接続を試みます。
　　　読み込む       ... 選択された行の設定を下部の「設定」枠内にロードします。
　　　名前を変更する ... 選択された行の設定の名前を変更します。
　　　削除           ... 選択された行の設定を削除します。

　設定 / 名前

　　設定リストで管理するための名前です。

　設定 / ホスト名

　　接続先サーバーのホスト名です。

　設定 / ポート番号

　　接続先サーバーのTCPポート番号です。

　設定 / 暗号化の選択

　　暗号化なし

　　　データを暗号化せずに通信を行います。
　　　この設定で、インターネットを介して接続するのは危険です。

　　鍵を使って暗号化

　　　通信データを鍵を使って暗号化します。

　　パスフレーズを使って暗号化

　　　通信データをパスフレーズを使って暗号化します。
　　　あまり安全ではないので、鍵を使った暗号化を推奨します。

　設定 / 鍵

　　暗号化に使用する鍵です。

　設定 / パスフレーズ

　　暗号化に使用するパスフレーズです。

　設定 / 中継用ポート番号

　　設定ダイアログの「中継用ポート番号」と同じです。

　設定を保存

　　「設定」枠内の内容を設定リストに追加又は上書きします。

　接続

　　「設定」枠内の内容で接続を試みます。

　キャンセル

　　このダイアログを閉じます。


■設定ダイアログ

　中継用ポート番号

　　通信内容の暗号化を行う場合に使う、暗号化トンネルが待ち受けに使用するTCPポート番号です。
　　クライアント環境で空いているTCPポートを指定して下さい。

　マウスカーソルがスクリーンの外に出てもアクティブにする。

　　これにチェックを入れると、
　　クライアントウィンドウ外でのマウス操作もサーバーに送信されるようになります。
