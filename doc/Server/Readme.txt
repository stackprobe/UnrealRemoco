===============================================================================

　UnrealRemoco / サーバー

===============================================================================


■起動方法

　UnrealServer.exe を実行して下さい。


■終了方法

　タスクアイコンを右クリックして "終了" を選択する。


■タスクアイコンの右クリックメニュー

　設定

　　設定ダイアログを開きます。
　　設定ダイアログを開いている間、クライアントから接続出来なくなります。

　再起動

　　内部的に再起動します。

　終了

　　このアプリケーションを終了します。


■設定ダイアログ

　ポート番号

　　接続先サーバーのTCPポート番号です。

　暗号化 / 暗号化の選択

　　暗号化なし

　　　データを暗号化せずに通信を行います。
　　　この設定で、インターネットを介して接続するのは危険です。

　　鍵を使って暗号化

　　　通信データを鍵を使って暗号化します。

　　パスフレーズを使って暗号化

　　　通信データをパスフレーズを使って暗号化します。
　　　あまり安全ではないので、鍵を使った暗号化を推奨します。

　暗号化 / 鍵

　　暗号化に使用する鍵です。

　暗号化 / パスフレーズ

　　暗号化に使用するパスフレーズです。

　暗号化 / 転送先ポート番号

　　通信内容の暗号化を行う場合に使う、暗号化トンネルの転送先に使用するTCPポート番号です。
　　サーバー環境で空いているTCPポートを指定して下さい。
